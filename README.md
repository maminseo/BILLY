 # 전공 물품 대여 앱 서비스 BILLY


1. 학교 인증 -> 학교 인증으로 사용자간 신뢰성 up

2. 학생대 학생 or 기업&대학 대 학생 간 물품 매칭

3. 대여한 물품에 대하여 ai를 통해 사용법이나 영상 크롤링

4. 파손이나 분실에 대하여 보증금 or 보험 설정으로 안전성 up

5. 쏘카처럼 빌려주기 전 사진촬영 -> 빌린 사람 또한 사진 촬영

- 확장성 

대학과의 협업으로 픽업박스 설치 -> 같은 학교 학생 간 대여률 상승
동네 대여점과 대학도 참여하여 학생들에게 대여 

학생이 소유하기에는 비싸거나 갯수가 적음( 수요는 많으나 공급은 적음) 
앱이 활성화 됨에 따라 우리가 직접 구매하여 학생들에게 대여 -> 수익성 향상

- 주요기능

장비 관리 및 신뢰 문제: 공유 과정에서 장비 파손, 분실, 먹튀와 같은 문제가 발생할 수 있습니다. 이를 방지하기 위한 안전 장치가 필요합니다.

신분 인증: 학생증 인증 등을 통해 서비스 이용자를 확실하게 파악해야 합니다.

평가 시스템: 이용자들이 서로를 평가할 수 있는 시스템을 도입하여 신뢰도를 높여야 합니다.

보증금 제도: 고가 장비에 대한 보증금 제도를 운영하여 파손이나 분실에 대비할 수 있습니다.

보험 및 손실 보상: 공유된 장비가 파손되거나 고장 났을 경우를 대비해 보험 상품을 연계하거나, 손실 보상 시스템을 마련해야 합니다.

편의성: 장비를 쉽게 등록하고, 필요한 장비를 효율적으로 검색할 수 있도록 사용자 인터페이스를 편리하게 만드는 것이 중요합니다.

활용성 낮은 장비: 특정 전공 외에는 수요가 적은 장비의 경우, 효율적인 거래를 보장하기 어려울 수 있습니다. 다양한 전공을 아우르면서도 수요가 꾸준한 장비를 중심으로 서비스를 시작하는 것이 좋습니다.
